# Integrador 01 - Bridge WebSocket <-> MQTT

Objetivo: recibir mensajes por WebSocket y publicarlos en MQTT.

## Tarea base
1. Levantar un servidor WebSocket.
2. Al recibir un mensaje, publicar en topic MQTT.
3. Mostrar en consola cada publicacion.

## Extension sugerida
- Agregar validacion de JSON.
- Agregar topic por aula/equipo.
- Guardar historial en memoria.
